# GrantProj
Итоговый проект. Java. Основы программирования на РТФ.
Вариант 4. 

Задания:
1. Обработать данные из CSV-файла в класс Гранта
2. Создать базу данных SQL Lite
3. Создать таблицу в базе данных и сохранить в ней набор данных Грантов
4. Выполнить 3 задачи
   4.1 Постройте график по среднему количеству рабочих мест для каждого фискального года
   4.2 Выведите в консоль средний размер гранта для бизнеса типа "Salon/Barbershop"
   4.3 Выведите в консоль тип бизнеса, предоставивший наибольшее количество рабочих мест, где размер гранта не превышает $55,000.00
5. Создать репозиторий с проектом
